# A10 Core Foundation

このフォルダは、A10理論群の共通基盤を整理するための公開候補資料である。

ここでのA10は、制約付き探索、モデル設計、評価ワークフローを整理するための **structured prior / geometric steering** として扱う。新しい自然法則、検証済み産業技術、完成した物理理論として提示するものではない。

## 含まれるもの

- A10基礎説明
- 主張境界
- AI支援開示
- 限界
- 実務位置づけ表
- citation metadata
- evaluation-only license notice

## フレームワーク・ビジュアル案内

初めて本リポジトリを見る技術的関心のある読者向けに、ブラウザだけで開けるフレームワーク案内ページを同梱しています。

`docs/framework_visual_orientation/index.html`

このページは、A10の基礎的な位置づけ、すなわち structured prior / geometric steering の反復ループ、制約下での mission-variable preservation、reduced surrogate の運用、robustness / tail-risk 評価、ドメイン横断時の限界、および claim boundary を短く整理するための補助資料です。

このページは説明補助であり、A10の妥当性を検証するものではありません。普遍的自然法則、産業実装、または安全認証を主張するものでもなく、各応用分野における個別の証拠や専門家評価を置き換えるものでもありません。

## 主張しないこと

- A10が普遍的自然法則として証明された
- 全A10応用が検証済みである
- 産業実装が可能な段階にある
- 物理的発見が確立した
- AI支援出力が専門家査読なしに真である

## 用途

このフォルダは、後続のA10応用論文・技術評価リポジトリの共通基盤として使う。専門家による批判的検証を前提とする。
