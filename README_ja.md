# A10 Core Foundation

このフォルダは、A10理論群の共通基盤を整理するための公開候補資料である。

ここでのA10は、制約付き探索、モデル設計、評価ワークフローを整理するための **structured prior / geometric steering** として扱う。新しい自然法則、検証済み産業技術、完成した物理理論として提示するものではない。

## 含まれるもの

- A10基礎説明
- 主張境界
- AI支援開示
- 限界
- 実務位置づけ表
- citation metadata
- evaluation-only license notice

## 主張しないこと

- A10が普遍的自然法則として証明された
- 全A10応用が検証済みである
- 産業実装が可能な段階にある
- 物理的発見が確立した
- AI支援出力が専門家査読なしに真である

## 用途

このフォルダは、後続のA10応用論文・技術評価リポジトリの共通基盤として使う。専門家による批判的検証を前提とする。
